# Main info

This is a command for 26.1 that gives you a diamond that is edible and has a custom name.

```
/give @a diamond[custom_name=[{"text":"Edible Diamond","italic":false}],lore=[[{"text":"1.21.11 btw, anyways EAT IT","italic":false}]],consumable={consume_seconds:2},food={nutrition:1,saturation:1}]
```

> [!IMPORTANT]
> This is only for 26.1 and 1.21.11. If used in any other version, it may not work.

> [!NOTE]
> You can change the diamond, description, eat time, nutrition and saturation, and name all you want
